## 分镜时间轴

```
中国水墨动画风格，15秒，9:16竖屏，空灵写意、孤冷出尘的深秋意境

0-3秒：【场景建立】- 远景缓慢推镜头，深秋枫叶林全景，层林尽染满山红遍。秋风
轻拂，枫叶如雨般簌簌飘落。白衣侠客背对镜头，静立于林间空地中央，衣袂随风微动，
水墨渲染的枫红与墨色交织，留白处营造空灵感

3-5秒：【角色亮相】- 中景环绕镜头（半周），从侧前方展示侠客。他面容沉静，目
光如深潭，右手缓缓握上剑柄，一寸寸将青锋拔出剑鞘。剑身上寒光流淌，映着飘落的
红叶。墨线勾勒轮廓，枫叶以淡彩胭脂红渲染

5-9秒：【剑招初展】- 中景快速推进，侠客身形骤动，长剑破空而出！剑气在地面卷起
落叶，身体在枫林中腾挪穿梭，剑光如银练飞舞，动作舒展而有力。水墨笔触的动势线
条跟随剑迹，墨色飞溅效果增强动感

9-11秒：【剑气高潮】- 近景跟拍+特写切换，侠客剑招大开大合，剑气激荡之下，
漫天枫叶被卷成旋涡状飞舞旋转。剑尖划过，一片枫叶被精准一分为二，叶脉纹路清晰
可见。墨色在此处最为浓重，枫叶红达到饱和度顶峰

11-13秒：【收招过渡】- 中景拉远，侠客收剑势，身体由动转静，剑尖画出一道优美
弧线后缓缓垂向地面。飘舞的枫叶随着剑气的消散逐渐减缓速度

13-15秒：【意境落版】- 全景逐渐拉远，侠客负手而立，长剑归鞘，漫天枫叶缓缓飘
落在肩头和地面。画面渐隐过渡到黑底白字：【字幕：枫林剑意 · 一剑知秋】，余韵
绵长

【声音】古琴独奏（悠远空灵）+ 秋风拂叶的环境音 + 剑锋破空的清越剑鸣声 + 枫叶
飘落的细微沙沙声

【参考】@图片1（C01）角色形象参考，@图片2（S01）枫叶林场景参考，@图片3（P01）
长剑造型参考
```

## 使用说明

### 在 Seedance 2.0 平台使用

1. **上传素材：**
   - 图片1（C01）：白衣侠客正面全身 → 用于角色形象参考
   - 图片2（S01）：枫叶林全景 → 用于场景参考
   - 图片3（P01）：三尺青锋剑 → 用于道具参考

2. **复制提示词：**
   将上方「分镜时间轴」中的完整提示词粘贴到 Seedance 2.0 输入框

3. **@引用素材：**
   在提示词末尾添加 `@图片1 角色形象参考，@图片2 场景参考，@图片3 剑造型参考`

4. **生成与调整：**
   - 如角色面部不一致，可单独用 C01/C02 生成角色图后再放入
   - 如动作过快或过慢，可微调时间轴分配
   - 建议开启 Seedance 高质量模式以获得最佳水墨渲染效果

### 提示词优化要点

| 优化方向 | 建议 |
|----------|------|
| 动作稳定性 | 动作描述中使用"缓缓""轻拂""慢"等词汇，避免剧烈旋转导致画面崩坏 |
| 风格一致性 | 所有描述中反复强调"Chinese ink wash painting style"确保风格统一 |
| 色彩控制 | 明确限定"ink black + rouge red"双色系统，避免AI自行添加过多颜色破坏水墨感 |
| 防崩约束 | 提示词中已包含"4K超高清""人物结构正常""动作流畅不僵硬"等约束词 |
| 留白处理 | 使用"negative space"关键词引导AI保留水墨画的留白意境 |
