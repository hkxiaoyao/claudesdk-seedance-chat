## 分镜时间轴

```
手绘水彩治愈动画风格，15秒，9:16竖屏，秋日离别、不舍却克制的情感氛围

0-2秒：【转场过渡】- 叠化转场，信纸上的墨迹幻化为旧式行李箱的轮子。镜头从车
轮缓缓上摇，水彩晕染效果过渡

2-4秒：【场景建立】- 全景，小镇车站月台，绿皮火车静静停靠，铁轨延伸向远方。
秋日午后的阳光带着淡淡的金色，远处的山峦在薄雾中若隐若现。站台老旧的招牌在风
中轻晃

4-7秒：【角色互动】- 中景，十八岁的阿欣背着大背包站在月台上，阿嫲站在她面前，
正低头往她包里塞一个鼓鼓的布袋。动作有些急切，怕她不要。
阿欣（无奈笑）：阿嫲，装不下了……
阿嫲（低头继续塞，声音轻轻的）：带着，城里买不到这个味儿。

7-9秒：【细节特写】- 近景特写，阿欣低头看向布袋，半透明的袋子隐约露出几罐腌
菜和两个水煮蛋，还微微冒着热气。阿欣的表情从无奈变为感动，喉头微动

9-11秒：【情感转折】- 中景，火车汽笛响起。阿嫲停下动作，直起身看着阿欣。两
人对视片刻，阿嫲嘴唇动了动，想说什么，最终只是点了点头。阿欣转身走向车厢

11-13秒：【情感高潮】- 近景，阿欣在车窗边坐下，回头寻找阿嫲的身影。窗外，阿
嫲站在月台上，花白的头发被秋风吹乱，她努力仰着头，朝车窗的方向笑着挥手，笑容
里带着不舍的颤抖

13-15秒：【结尾落版】- 大特写，火车缓缓启动，阿嫲脸上的笑容慢慢淡去，眼角一
点泪光在阳光下闪烁，但始终没有让它落下。她只是站在原地，目送火车越走越远。
→ 画面如水彩晕染模糊，叠化转回书桌
→ 【水滴落在信纸上的声音】
→ 特写，一滴泪落在信纸上，墨迹缓缓晕染开来

【声音】悠扬的口琴旋律（略带忧伤）+ 绿皮火车的汽笛声 + 车轮滚动声渐强 + 秋风
声 + 一滴泪水落在纸上的清晰声响

【参考】@图片1（C06）18岁阿欣形象参考，@图片2（C03）阿嫲形象参考，@图片3（S03）
小镇车站月台场景参考
```
