## 四、风格参考说明

| 类别 | 关键词 |
|------|--------|
| **核心风格** | Hand-painted watercolor animation style（手绘水彩动画风格） |
| **渲染方式** | soft watercolor texture（柔和水彩纹理），wet-on-wet blending（湿画法晕染） |
| **线条风格** | delicate ink outlines（细腻墨线勾勒），slightly loose edges（边缘略带自然晕散） |
| **色彩体系** | 暖色调为主：cream white（米白）+ warm amber（暖琥珀）+ dusty rose（灰粉）+ sage green（鼠尾草绿）+ denim blue（靛蓝） |
| **光线氛围** | warm lamplight（温暖灯光），golden hour sunlight（金色夕阳），dappled light（斑驳树影） |
| **画质保障** | 4K ultra HD, details clear, smooth animation, no flicker |
| **防崩约束** | character structure natural, face consistent across shots, facial features stable, proportions correct, motion smooth not stiff |
